## Project Setup

Clone the Project 
```bash
git clone https://github.com/Ahmedashref26/Chirper.git
``` 
Install the dependencies 
```bash
npm install
```

## Usage

Start the development server.

```bash
npm start
```

View app at

- [http://localhost:3000](http://localhost:3000)

What you can do with this application

- search for recipes.
- bookmark recipes.
- upload recipes
